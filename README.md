# 🔳 Text to QR Code Generator

A modern and responsive web application that converts any text or URL into a QR code instantly. This project is built using HTML, CSS, and JavaScript and provides a fast, clean, and user-friendly experience.

---

## 🚀 Features

* 🔤 Convert text or URLs into QR codes instantly
* ⚡ Real-time QR generation
* 📱 Fully responsive design (mobile + desktop)
* 🎨 Clean and minimal UI
* 🖼️ High-quality QR output

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* QR Code Library / API

---

## 📂 Project Structure

```
qr-generator/
│── index.html
│── style.css
│── script.js (or app.js)
│── README.md
```

---

## ⚙️ How to Use

1. Enter text or URL in the input field
2. Click on **Generate QR Code**
3. Your QR code will appear instantly
4. Scan and use it easily
